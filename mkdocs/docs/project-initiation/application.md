# Leave Management System

- **Type:** web_application
- **Domain:** 
- **Description:** A system that enables employees to submit leave requests, managers to approve or reject them, and HR admins to oversee leave types, balances, and history.

## Actors

- **Employee** — Submits leave requests, views own balances and history, modifies or cancels own requests
- **Manager** — Approves or rejects leave requests for direct reports, adds comments
- **HR Admin** — Manages leave types and balances, oversees all requests and history, maintains system configuration
