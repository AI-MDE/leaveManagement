# Assess Methodology

| | |
|---|---|
| **Name** | `assess_methodology` |
| **Phase** | governance |
| **Intent** | validate |
| **Calls** |  |

## Rules

- Validate the chosen methodology against known patterns and industry best practices
- Seek and cite authoritative references (academic, industry, standards bodies) that support or challenge the choice
- Identify risks or gaps in the chosen methodology relative to the project domain and scale
- Recommend adjustments if warranted — do not impose, present as options
- Record findings in methodology-assessment.md with references and reasoning
