# Assess Architecture

| | |
|---|---|
| **Name** | `assess_architecture` |
| **Phase** | governance |
| **Intent** | validate |
| **Calls** |  |

## Rules

- Validate the chosen architecture against the application requirements, domain constraints, and external references
- Seek and cite authoritative references that support or challenge the architectural decision
- Evaluate against quality attributes: scalability, maintainability, security, resilience, cost
- Identify risks, known failure modes, and mitigations for the chosen pattern
- Recommend adjustments if warranted — do not impose, present as options
- Record findings in architecture-assessment.md with references and reasoning
